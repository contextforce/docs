## Key Features

### 1. Content Extraction 🔍
LEXI extracts various elements from YouTube videos, including complete transcripts. Users can ask specific questions about the video content and receive accurate answers based on the extracted information.

### 2. Interactive Transcript 📝
<Card img="/images/youtube/lexi-youtube-transcript.jpg">
- • Access the full video transcript via the Transcript icon
- • Click any section to navigate directly to that timestamp
- • Follow along with video content in text form
</Card>

### 3. Smart Prompts 🧠
<Card img="/images/youtube/lexi-youtube-settings.jpg">
- • Create specialized prompts for YouTube content analysis
- • Access video metadata including title, description, and transcript
- • Define custom analysis patterns for consistent results
</Card>

### 4. One-Click Summary ⚡
<Card img="/images/youtube/lexi-youtube-summarize.jpg">
- • Access instant video analysis via the interface button
- • Generate quick summaries of video content
- • Toggle functionality through Web Assistant settings
</Card>

### 5. Knowledge Base 📚
<Card img="/images/youtube/lexi-youtube-memo.jpg">
- • Save videos to your searchable knowledge base
- • Reference saved content using the "@" symbol
- • Build a searchable library of video content
- • Quick access to saved video information
</Card>

> **Example workflow:**
> 1. Load a video into your knowledge base in the MEMO Tab
> 2. Use the "@" symbol in chat to reference and summarize saved videos
> 3. Build a comprehensive collection of valuable content for future reference!
